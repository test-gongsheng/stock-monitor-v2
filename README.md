# 股票投资监控系统 v2.0

基于中轴价格仓位控制法的股票投资监控平台，支持A股和港股实时监控。

## 投资框架

- **总投资额度**: 800万元人民币
- **A股投资上限**: 50万元/只（个别可达100万元）
- **港股投资上限**: 150万元/只
- **核心策略**: 中轴价格仓位控制法 + 个性化网格策略
- **信息来源**: 财联社（优先）、东方财富、同花顺

## 核心策略

### 基础策略：中轴价格仓位控制法

**仓位结构：**
- 底仓：50%（长期持有）
- 浮动仓：50%（动态调整）

**操作规则：**
- 股价较中轴上涨8% → 减持浮动仓20%
- 股价较中轴下跌8% → 增持浮动仓20%

### 进阶策略：个性化网格策略

针对特定股票可定制参数，如阿里巴巴港股：
- 底仓比例：60%
- 浮动仓比例：40%
- 买入触发：下跌5%买入10万
- 卖出触发：上涨8%卖出10万

## 监控界面（四大区域）

1. **自选股票显示区** - 实时持仓、中轴价格、买卖触发点
2. **个股情绪跟踪区** - 财联社新闻、机构评级、资金流向
3. **市场热点跟踪区** - 科技板块、金属类板块监控
4. **实时提醒区** - 买卖触发、新闻推送、风险预警

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Python Flask
- **UI框架**: 原生实现，深色主题

## 快速开始

### 安装依赖

```bash
pip install flask flask-cors
```

### 启动系统

```bash
./start.sh
```

或

```bash
python3 app.py
```

### 访问系统

浏览器打开: http://localhost:5000

## 功能截图
